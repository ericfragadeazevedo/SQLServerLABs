# 🧪 SQL Server Labs | Estudos e Práticas

Repositório dedicado aos meus estudos práticos em **SQL Server**, com foco em administração, performance, alta disponibilidade e cenários reais de banco de dados.

---

## 🎯 Objetivo

Consolidar conhecimentos práticos em **Administração de Banco de Dados**, com base nos estudos para certificações Microsoft:

* DP-300 (Azure Database Administrator)
* DP-800 (SQL Server Híbrido)
* AZ-104 (Azure Administrator)

---

## 🧠 Abordagem

Cada LAB deste repositório segue uma abordagem prática:

* 🔍 Cenário real ou simulado
* ⚙️ Execução técnica passo a passo
* 📊 Análise de resultado
* 💡 Lições aprendidas

### 💾 Backup e Restore

* Tipos de backup (Full, Differential, Log)
* Estratégias de recuperação
* Testes de restore

* Análise de queries lentas
* Uso de Execution Plans
* Identificação de gargalos

### 📊 Indexação

* Criação e análise de índices
* Impacto em performance

* Logins e usuários

* Boas práticas de acesso

### ⚙️ Alta Disponibilidade

* AlwaysOn (conceitos)
* Failover
* Continuidade de serviço
